# StreamVault 3.0.0

**Stable suite release** — pairs with [StepDaddy Gateway 3.0.0](https://github.com/thothassistantai-web/stepdaddy-gateway-android/releases/tag/v3.0.0).

## Highlights

- StepDaddy Gateway plugin: `streamvault.m3u` + `epg.xml` one-tap import
- Gateway lifecycle auto-start and 15-minute recovery worker
- Resume last live channel toggle
- Browse/guide caches for large gateway catalogs
- Special event display names (health dots stripped in UI)
- Touch playback gestures on phones and tablets

## Install

Sideload `streamvault-3.0.0-release.apk` (`com.streamvault.app`).

## Setup

Plugins → StepDaddy Gateway → Test connection → Add provider.
